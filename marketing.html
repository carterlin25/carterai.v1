行銷文案自動化

以 GPT-4 生成行銷文案與報表，支援 A/B 測試與自動發佈。

專案簡介

本專案結合 GPT 模型與社群平台 API，讓行銷團隊可快速生成多版本文案， 同時追蹤貼文表現數據，定期生成成效報表，實現「一鍵自動化行銷」。

功能特色

  • ✍️ 自動產出 5 種風格文案（正式、幽默、感性、技術、促銷）
  • 📊 A/B 測試互動數據分析
  • 🕓 自動排程貼文發佈
  • 📧 每週寄送成效報告至 Email

技術架構

  • Backend：Python + Flask
  • API：OpenAI GPT-4 + Meta Graph API
  • Frontend：React Dashboard
  • 資料儲存：Google Sheets API
  • 通知：LINE Notify
