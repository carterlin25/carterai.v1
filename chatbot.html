智能客服系統（Demo）

整合企業 FAQ 與 GPT-4，打造能即時理解與回覆客戶問題的 AI 客服。

專案簡介

本專案以 RAG（Retrieval-Augmented Generation）為核心，結合 GPT-4 模型， 能從企業內部文件與 FAQ 資料中擷取相關答案， 透過 Webhook 連動至網站或 LINE 官方帳號，提供智慧化客服體驗。

功能特色

  • 📚 即時回答客戶常見問題，並附上資料來源連結
  • 🔗 支援 Webhook 整合 Line / 網站客服模組
  • 🧩 可動態更新知識庫（上傳 PDF 或 Google Sheet）
  • 📈 提供問答紀錄統計分析

技術架構

  • Frontend：Next.js + Tailwind CSS
  • Backend：FastAPI + PostgreSQL
  • AI 模型：OpenAI GPT-4 + LangChain
  • 資料來源：Google Sheet / PDF / JSON
  • 部署：Vercel + Cloudflare Worker
